# my-tunes-laravel-rest-api
laravel as a rest api for angular/vue &amp; react

# packages
    jwt-auth // auth for rest-api

# start
    install composer
    install composer/laravel
    composer update
    php artisan serve

## generate json-web-token (jwt)
    php artisan jwt:generate
